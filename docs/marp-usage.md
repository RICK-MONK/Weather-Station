# Marp Usage For This Project

Use [weather-station-marp.md](./weather-station-marp.md) as the presentation source.

## Recommended workflow

1. Install the **Marp for VS Code** extension.
2. Open `docs/weather-station-marp.md`.
3. Run `Marp: Open Preview to the Side` in VS Code.
4. Edit the slide content as needed.
5. Run `Marp: Export Slide Deck...` to export as PDF, HTML, or PowerPoint.

## What to customize before presenting

- your name
- project date
- project photos
- dashboard screenshots
- any final measured results you want to show

## Good visuals to add

- PCB sensor board photo
- PCB display/gateway board photo
- dashboard screenshot
- analysis page screenshot
- serial monitor output showing successful backend update
